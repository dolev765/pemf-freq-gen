# Magnetometer Module (Extracted from phyphox-android)

This is a simplified, standalone magnetometer module extracted from the phyphox-android project.

## Files

- `MagnetometerInput.java` - Main magnetometer sensor input class
- `DataBuffer.java` - Data buffer for storing sensor readings
- `ExperimentTimeReference.java` - Time reference handling

## Usage

The magnetometer module provides:
- Reading magnetic field data in microteslas (ÂµT)
- Support for both calibrated and uncalibrated magnetometer
- X, Y, Z axis readings
- Absolute value calculation
- Timestamp tracking
